# 简单选择排序

**思路分析**

1.在第i趟排序中，从array[i]开始，到array[ n - 1 ]中选出被排序数的最小的一个，将该数放到array[ i - 1 ]。
2.i+1，继续下一步，直到i为n - 1。

```c
void Swamp( int &a, int &b )
{
	int temp = a;
	a = b;
	b = temp;
}

int Compare(int a, int b)
{
	if( a == b )
		return 0;
	if( a < b )
		return -1;
	else 
		return 1;
}

template<class T>
void selectSort( T A[], int n )
{
	for( int i = 0; i < n - 1; i++ )
	{
		//先假设第i个为最小
		int small = i;
		for( int j = i + 1; j < n; j++)
		{
			if( Compare(A[j], A[small]) == -1 ) 
				small = j;
		}
		Swamp(A[small],A[i]);
	}
}
```

时间复杂度O(n^2)。
